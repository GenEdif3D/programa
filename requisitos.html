Requerimientos del Sistema

Para garantizar el correcto funcionamiento del software Gen-Edif3D, se deben cumplir las siguientes especificaciones mínimas:

  • Sistema Operativo: Windows 11, Windows 10 (versión 1909 o superior), Windows Server 2019
  • Arquitectura: 32 y 64 bits
  • Procesador: Intel o AMD x86-64
  • Memoria RAM: Mínimo 4 GB, recomendado 8 GB
  • Espacio en Disco: Al menos 1 GB de almacenamiento libre
  • Software adicional requerido: Microsoft Excel®, ETABS® versión 18.1.1
